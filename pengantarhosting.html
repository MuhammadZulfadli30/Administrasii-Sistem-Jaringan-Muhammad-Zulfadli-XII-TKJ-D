Pengantar Hosting

Pahami dasar-dasar tentang hosting, serta perbedaan layanan GitHub Pages dan Vercel.

Pengertian Hosting

Hosting adalah layanan yang menyediakan ruang di server untuk menyimpan file website agar bisa diakses melalui internet. Semua file seperti HTML, CSS, gambar, dan video tersimpan di server yang selalu aktif.

Hosting ibarat bangunan dari sebuah rumah, sedangkan domain adalah alamatnya. Terdapat berbagai jenis hosting seperti shared hosting, cloud hosting, VPS, dan dedicated server.

GitHub Pages dan Vercel

GitHub Pages

GitHub Pages adalah layanan hosting gratis untuk website statis — cocok untuk portofolio dan dokumentasi.

Kelebihan: cepat & gratis. Kekurangan: tanpa backend.

Vercel

Vercel adalah platform PaaS dengan auto deployment dan integrasi CI/CD.

Kelebihan: performa tinggi. Kekurangan: berbayar untuk fitur lanjutan.

Jenis-jenis Hosting

Shared Hosting

Situs berbagi sumber daya server (CPU/RAM/disk) dengan pengguna lain.

Kelebihan: murah & mudah. Kekurangan: performa dipengaruhi pengguna lain.

VPS (Virtual Private Server)

Lingkungan server virtual terisolasi pada mesin fisik yang sama.

Kelebihan: kontrol & sumber daya terjamin. Kekurangan: butuh setup teknis.

Cloud Hosting

Memanfaatkan beberapa server terdistribusi — scalable & fault-tolerant.

Kelebihan: skalabilitas & uptime. Kekurangan: model harga bisa kompleks.

Dedicated Server

Sebuah server fisik dipakai penuh oleh satu pengguna/organisasi.

Kelebihan: performa maksimal. Kekurangan: mahal & perlu manajemen.

Reseller Hosting

Jual kembali paket hosting kepada klien Anda dengan panel terpisah.

Kelebihan: peluang bisnis. Kekurangan: support & pembagian sumber daya.

Managed WordPress

Hosting dioptimalkan untuk WordPress: auto-update, backup, dan keamanan.

Kelebihan: siap-pakai & aman. Kekurangan: biaya & terbatas ke WP.

Static Hosting (GitHub Pages)

Untuk situs statis (HTML/CSS/JS). Cepat dan sering gratis.

Kelebihan: cepat & murah. Kekurangan: tidak mendukung backend.

PaaS (Platform as a Service) — contoh: Vercel

Platform yang memudahkan deploy dengan build & scaling otomatis.

Kelebihan: CI/CD & performance. Kekurangan: biaya untuk fitur lanjutan.

Kesimpulan

Pilih hosting berdasarkan kebutuhan: hemat & sederhana (Shared/Static), kontrol (VPS/Dedicated), atau kemudahan deploy (PaaS).
